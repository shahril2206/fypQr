Abstract:

The "EventKamek: Event Vendor Management System" is a web-based platform designed to streamline the event management process, particularly focusing on vendor slot booking. This system addresses inefficiencies in traditional methods, such as fragmented tools and manual workflows, by offering a centralized, user-friendly solution. With features like real-time slot updates, categorized vendor slots, and comprehensive event management tools, the platform fosters collaboration between organizers and vendors. The project employs the Rapid Application Development (RAD) methodology, ensuring iterative development and stakeholder involvement. Aligned with Sustainable Development Goal 8, the system promotes innovation, entrepreneurship, and economic growth. Its functionalities include event creation, vendor slot management, notifications, and an intuitive calendar/map view for better user experience. This platform enhances event organization, vendor engagement, and community connectivity, providing a scalable solution for the Kuching area and beyond.
